It looks like your Github account has your email address set as private. If you'd like to continue with a Github login you'll have to fix this in order to use LatterDayWard.com.

There are 2 places you'll have to check at Github.com.

First check and make sure the checkbox for setting your email private is unchecked in GitHub Email settings

After unchecking that option, you'll then need to select an email that you want public in your GitHub Profile settings
